CropIQ

CropIQ is an AI-powered crop recommendation platform that helps farmers select the most suitable crops based on soil health, environmental conditions, and budget. Using Machine Learning and real-time weather data, the system analyzes factors such as Nitrogen, Phosphorus, Potassium, pH, temperature, humidity, rainfall, location, and budget to recommend the top three crops with confidence scores. Built with React + Vite and FastAPI, the platform integrates OpenStreetMap and Open-Meteo APIs to automatically fetch the user's location and live weather conditions. A Random Forest Classifier powers the recommendation engine, while Pydantic ensures secure data validation. Along with crop predictions, CropIQ provides practical farming insights such as water requirements and cultivation recommendations, making it a smart and scalable solution for precision agriculture.
